# Коттедж «Бурон Сердце» — сайт аренды

Одностраничный сайт для аренды коттеджа в горах Северной Осетии, посёлок Бурон.
Тёплый атмосферный дизайн, онлайн-бронирование и отправка заявок хозяину.

🔗 **Живой сайт:** https://buron-serdtse.netlify.app

## Возможности

- Бронирование на одни или несколько суток с автоматическим расчётом стоимости
- Отправка заявок на почту (имя, телефон, даты, число гостей)
- Галерея, блок удобств, правила проживания
- Карта расположения и описание маршрута
- Адаптивная вёрстка — корректно смотрится на телефоне, планшете и компьютере

## Технологии

- HTML
- CSS
- JavaScript
